# number-game

The only file worth anything is fours.hs

The solve function takes a list of numbers and a single number and returns all the ways that list can be combined with basic arithmetic into that single number.

Input: [1,2,3,4] 10

output: 1 + 2 + 3 + 4 = 10
     1 * 2 * 3 + 4 = 10
     and so on....
